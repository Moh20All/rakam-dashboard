00 — Getting Started

First Launch & Activation

بدء التشغيل والتفعيل

Rakam is a premium, licensed tool. Every installation is cryptographically bound to the hardware it runs on.

License Activation
1
Launch the application. A unique Machine ID is generated from your hardware.
2
Enter your Activation Key provided after purchase.
3
The key is verified and bound to your machine. One key = one machine.
Setup Wizard (3 Steps)
1
Enter your Store Name — shown on all receipts and reports.
2
Select your Currency: DA (Algerian Dinar), EUR, or USD.
3
Create your Administrator Account — username and secure password.
🔒 Security Note Your Machine ID is derived from hardware identifiers. If you replace your machine, contact support for a key transfer.

02 — Repair Module

Repair Tracking

متابعة الإصلاحات

The heart of Rakam. Every device that enters your shop is tracked from check-in to collection.

📋
Check-In Wizard
Capture the customer name, phone, device model, IMEI number, visible faults description, and estimated repair cost in a guided multi-step form. Print the customer's copy immediately.
🔄
Status Lifecycle
Every repair moves through clear stages: Pending → In Progress → Ready → Collected. Visual markers on the dashboard give you an instant overview of your entire workshop load.
🛡️
Warranty Tracking
Rakam automatically calculates and monitors the warranty period for each completed repair. When a device comes back within warranty, the system flags it immediately.
💡 Tip Always enter the IMEI during Check-In. It acts as a unique identifier and helps resolve disputes with customers.
03 — POS Module

POS & Inventory

نقطة البيع والمخزون
📷
Barcode Scanning
Connect any USB barcode scanner. Scan a part's barcode to instantly add it to a sale or restock it. No manual entry, no errors.
📱
Trade-In & IMEI
Accept used devices as trade-ins. Assign a condition grade, record the IMEI, and Rakam tracks it as inventory. Sell it later directly through POS.
⚠️
Low Stock Alerts
Set a minimum threshold per item. When stock drops below it, Rakam alerts you on the dashboard. Never be caught without a screen or battery during a repair.
04 — Financial Module

Financial Intelligence

الذكاء المالي
📅
Daily Close — Z-Report
At the end of each day, open the Daily Close screen. Rakam tallies all sales, repairs, and expenses. Hit Close Day to seal the record and calculate exact cash on hand.
📈
Net Profit Analytics
Real-time revenue versus cost tracking. See your profit margin on every repair and sale. Identify your most and least profitable categories at a glance.
🗓️
Monthly Summaries
Elegant charts reveal sales trends across months. Compare periods, spot seasonality, and make smarter purchasing decisions backed by your own real data.
⭐ Best Practice Run the Daily Close every evening without exception. This creates an unbreakable historical record and makes monthly reports perfectly accurate.
05 — Daily Use

A Typical Day
with Rakam

يوم عمل نموذجي مع رقم

Follow this workflow to get the most out of Rakam every day.

I
Morning
Review the Dashboard
Open Rakam. Check the "Repairs Pending" section to see what's waiting. Review any low-stock alerts that appeared overnight.
II
Customer Arrives
Check-In a Device
Use the Check-In Wizard to log the ticket. Capture customer details, IMEI, fault description, and estimated cost. Print the receipt for the customer to keep as proof.
III
During Repair
Update Status to "In Progress"
As you begin work, update the repair status. If you need to source a part, note it in the ticket. The customer's device is always traceable.
IV
Repair Complete
Mark as "Ready" → "Collected"
Once fixed, mark the ticket "Ready". When the customer arrives and pays, mark it "Collected". Rakam logs the warranty period automatically from this moment.
V
Walk-In Sales
Use the POS
Scan barcodes or search items to add them to the cart. Select the payment method — cash, card, or partial. Print the receipt instantly via thermal printer.
VI
End of Day
Run the Daily Close
Navigate to Daily Close. Verify the earnings summary matches your physical cash. Hit "Close Day". Rakam seals the day's performance permanently into history.
06 — Developer

Technical Setup

الإعداد التقني

For developers building or modifying Rakam. Built with Electron, React, and Prisma over SQLite.

Installation
1
Prerequisites: Node.js v20+ and npm or yarn.
2
Clone the repository and run npm install.
3
Create .env files in the root and in rakam-dashboard.
4
Generate DB: npm run prisma:generate then npm run prisma:push.
Build & Package
→
Run npm run electron:build to generate the Windows installer.
→
The .exe installer is output to the /release directory.
→
Tech stack: Electron · React · Prisma · SQLite (offline-first).
